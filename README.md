# StartupForge

StartupForge is a modern startup collaboration platform where founders can build their startup teams by posting opportunities, and collaborators can discover startups, apply for roles, and track their application status. The platform features secure authentication, role-based dashboards, Stripe payment integration for premium features, and a responsive user experience.

## Live Site

https://startup-forge-wheat.vercel.app

## Features

- Better Auth Authentication (Email & Google)
- Founder, Collaborator & Admin Dashboards
- Startup & Opportunity Management
- Application Tracking System
- Stripe Payment Integration
- JWT Authentication with HTTPOnly Cookies
- Search, Filter & Pagination
- Fully Responsive Design
- Framer Motion Animation

## Tech Stack

**Frontend:** Next.js, React, Tailwind CSS, HeroUI

**Backend:** Node.js, Express.js, MongoDB, Mongoose, Better Auth, JWT, Stripe
